# Customer Support Analytics

## Project Overview

This project analyzes customer support data to identify trends, measure performance, and provide business recommendations.

## Tools Used

- SQL
- Python (Pandas)
- Microsoft Excel
- Power BI

## Business Questions

1. How many tickets were handled?
2. Which agent handled the most tickets?
3. Which region generated the most tickets?
4. Which issue type was most common?
5. What was the average resolution time?
6. What was the average customer satisfaction?

## Key Insights

- Ahmed handled the highest number of tickets.
- Addis generated the highest ticket volume.
- Network issues were the most common.
- Billing issues required the longest resolution time.
- Addis achieved the highest customer satisfaction.
