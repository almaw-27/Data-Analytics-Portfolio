# Executive Summary

This project analyzes customer support operations using SQL and Python.

The analysis focused on ticket volume, issue types, customer satisfaction, and agent performance.

## Results

- Total Tickets: 10
- Average Resolution Time: 27 minutes
- Average Satisfaction: 4.0/5
- Top Agent: Ahmed
- Top Region: Addis
- Most Common Issue: Network
- Slowest Issue: Billing

These insights can help managers improve operational efficiency and customer satisfaction.
