# Sales Performance Analytics

## Project Overview

This project analyzes sales performance across products, regions, and sales representatives.

## Tools Used

- SQL
- Python (Pandas)
- Excel
- Power BI

## Business Questions

1. What are total sales?
2. Which product generated the highest revenue?
3. Which region has the highest sales?
4. Who is the best-performing salesperson?
5. Which category is most profitable?
6. What are the monthly sales trends?

## Key Insights

- Laptop generated the highest revenue.
- Addis had the highest sales.
- John was the top-performing salesperson.
- Electronics was the most profitable category.
- Sales totaled 101,000 ETB.
