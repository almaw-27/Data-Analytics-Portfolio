# HR Analytics

## Project Overview

This project analyzes employee data to understand workforce distribution, salaries, and department performance.

## Tools Used

- SQL
- Python (Pandas)
- Excel
- Power BI

## Business Questions

1. Which department has the highest average salary?
2. Which department has the most employees?
3. Who earns the highest salary?
4. What is the average salary across the company?
5. How are employees distributed across departments?

## Key Insights

- IT has the highest average salary.
- IT has the largest number of employees.
- Meron has the highest salary.
- Average salary is competitive across departments.
